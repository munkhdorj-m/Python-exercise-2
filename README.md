# Python IF/ELSE

---

## Exercise 1

**Problem:**

Find the price of the cinema ticket based on age. If age is under 3 years old, the ticket is free. If ages 3-12, tickets are $10. If 65 or older, tickets are $12. Tickets for all other ages are $15.

**Example:**

| Input | Output |
|-------|--------|
| 2     | 0      |
| 5     | 10     |
| 30    | 15     |
| 70    | 12     |
